## 🧰 Your Complete grep Arsenal

```
🔍 Basic Search:
   ├── grep "text" file → Find text in files
   ├── grep -i "text" → Case-insensitive search  
   └── grep -c "pattern" → Count matches

🎯 Advanced Options:
   ├── grep -n → Show line numbers
   ├── grep -v → Exclude matches
   ├── grep "^start" → Lines beginning with text
   └── grep "end$" → Lines ending with text

⚡ Pipeline Mastery:
   ├── cmd | grep pattern → Filter outputs
   ├── grep term1 | grep term2 → Multi-stage filtering
   └── grep -E "term1|term2" → OR searches
```
