## 🔍 Practice Exercise 1: Understand Absolute vs Relative Paths

Let's learn the difference between full addresses and relative directions:

```bash
cd /tmp && pwd
```{{exec}}

```bash
cd ../etc && pwd
```{{exec}}

**What we're doing:** Using absolute path `/tmp` (full address from root), then relative path `../etc` (directions from current location)  
**What you'll see:** How absolute paths start with `/` and give exact locations, while relative paths use `..` for navigation from where you are! 🗺️

---

## 🔍 Practice Exercise 2: Practice Complex Multi-Level Navigation

Now let's practice navigating through multiple directory levels efficiently:

```bash
cd ~/workspace/projects && pwd
```{{exec}}

```bash
cd ../../ && pwd
```{{exec}}

**What we're doing:** Jumping deep into your project structure, then using `../..` to go back up two levels at once  
**What you'll see:** How you can efficiently navigate complex directory trees without typing long paths! 🎢

## 🔍 Practice Exercise 3: Master the Professional Navigation Workflow

Let's practice the complete workflow that Linux experts use every day:

```bash
pwd && ls -la && cd /tmp && pwd && ls | head -3
```{{exec}}

**What we're doing:** Check location → see contents → move to new location → confirm position → explore new area  
**What you'll see:** A complete professional navigation sequence that gives you total awareness and control! 🧭
