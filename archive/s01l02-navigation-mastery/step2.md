# Step 2 — Navigate Like a Digital Explorer 🧭

Great job exploring! Now let's learn to move around the file system like a pro. Think of `cd` as your teleportation device! ✨

## 🔍 Practice Exercise 1: Enter and Explore Directories

Let's start by moving into one of your directories and seeing what's inside:

```bash
cd projects && pwd && ls
```{{exec}}

**What we're doing:** Changing to the projects directory, confirming location, and listing contents  
**What you'll see:** You've teleported into the projects folder and can see what's there - like walking into a room and looking around! 🚪

---

## 🔍 Practice Exercise 2: Master Navigation Shortcuts

Now let's learn some super useful shortcuts that will save you tons of typing:

```bash
cd ~ && pwd
```{{exec}}

```bash
cd - && pwd
```{{exec}}

**What we're doing:** Using `~` to go home instantly, then `-` to go back to previous location  
**What you'll see:** `~` is your "home" button and `-` is your "back" button - instant navigation! 🏠

---

## 🔍 Practice Exercise 3: Practice Moving Up the Directory Tree

Let's learn to navigate upward through the directory structure:

```bash
cd /usr/bin && pwd
```{{exec}}

```bash
cd ../.. && pwd
```{{exec}}

**What we're doing:** Going to `/usr/bin`, then using `../..` to go up two directory levels  
**What you'll see:** How `..` means "parent directory" and you can chain them to go up multiple levels! ⬆️
