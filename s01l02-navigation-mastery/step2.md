## Enter a Directory

```bash
cd projects && pwd
```{{exec}}

**You teleported!** Location changed instantly.

---

## Go Back Up

```bash
cd .. && pwd
```{{exec}}

**The `..` means parent directory** - one level up.

---

## Jump Home Instantly

```bash
cd ~ && pwd
```{{exec}}

**The `~` is your home shortcut** - works from anywhere!

---

## Visit System Root

```bash
cd / && ls
```{{exec}}

**Welcome to root!** The foundation of Linux.

---

## Tab Completion Magic

```bash
cd ~/workspace/proj
```

**Press TAB** (don't hit Enter yet!) - Linux completes "projects"!

```bash
cd ~/workspace/projects
```{{exec}}

---

## Navigation Shortcuts

```bash
cd ~/workspace/documents/reports && pwd
```{{exec}}

**Jump multiple levels** in one command!
